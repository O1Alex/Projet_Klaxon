# Projet_Klaxon

Afin de pouvoir lancer se projet il faudra au préalable installer PHP sur votre machine et télécharger un environnement de développement comme XAMPP par exemple.

Une fois cela fais il faudra copier le projet dans le dossier htdocs présent dans le dossier XAMPP.

Pour créer la base de donnée et l'alimenter, il faudra lancer MySQL et Appache depuis le tableau de bord de xampp puis aller sur admin, situé en face de la ligne MySQL taper l'identifiant et le mot de passe présent dans le fichier database.php.
Dans un premier temps il faudra créer une base de données que vous nommerez "covoiturage" puis importer le fichier databasebase.php afin de créer la base de données et dans un second temps database_data.php afin de l'alimenter.

Pour lancer le projet il faudra ensuite lancer MySQL et Appache et dans la barre d'adresse de votre navigateur tapper localhost/Projet_Klaxon/View/homepage.php

Il y aura 3 choix une fois sur cette page. La voir en tant que visiteur, se connecter en tant qu'utilisateur ou se connecter en tant qu'administrateur.
